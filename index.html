Super API

Cosas variadas

  • GET /api/meaningoflife para obtener la respuesta al sentido de la vida como un string
  • GET /api/bands para obtener la lista de grupos como un array de object con tres string
  • GET /api/trapi para obtener la lista de cantantes de trap como un array de object con tres string

Usuarios

  • GET /api/users para obtener la lista completa de usuarios
  • POST /api/register para añadir un nuevo usuario con los parámetros "user" y "pass" de tipo string
  • POST /api/login para loguear un usuario con los parámetros "user" y "pass" de tipo string

Bases de datos

Tonterías

  • GET /api/db/meaningoflife para obtener la respuesta al sentido de la vida como un string
  • GET /api/db/tonterias para obtener la lista completa de tonterías como un array de object con un string
  • POST /api/db/tonterias para añadir una nueva tontería con el parámetro "quote" de tipo string

Cumpleaños

  • GET /api/db/bdays para obtener la lista de cumpleaños como un array de object con dos string y un number
  • POST /api/db/bdays para añadir un nuevo cumpleaños con los parámetros "name" y "image" de tipo string y "date" de tipo number

Eventos

  • GET /api/db/events para obtener la lista de eventos como un array de object con un string y un number
  • POST /api/db/events para añadir un nuevo evento con el parámetro "name" de tipo string y "date" de tipo number
